# Python Turtle 海龟画图练习题

本题集用于练习 Python 标准库中的 `turtle` 海龟画图模块。

题目按照难度由浅入深排列，主要训练：

- 海龟移动与转向
- 循环绘图
- 颜色和填充
- 函数封装
- 坐标定位
- 随机数
- 键盘事件
- 简单动画与游戏逻辑

> 建议每道题单独建立一个 `.py` 文件。  
> 先独立完成，再检查代码是否可以进一步使用循环或函数简化。

### 建议文件名

```text

```

### 题目要求

绘制 5 个同心圆，半径依次为：

```text
20、40、60、80、100
```

要求：

1. 使用循环；
2. 所有圆的圆心必须相同；
3. 每个圆之间不能出现多余连接线；

`turtle.circle(r)` 并不是以海龟当前坐标为圆心绘制圆。

## 题目 15：键盘控制海龟

### 建议文件名

```text
15_keyboard_control.py
```

### 题目要求

编写一个可以使用键盘控制海龟移动和改变颜色的程序。

### 操作规则

| 按键 | 功能 |
|---|---|
| `↑` | 前进 20 像素 |
| `↓` | 后退 20 像素 |
| `←` | 左转 15° |
| `→` | 右转 15° |
| 空格键 | 清空画布 |
| `r` | 将画笔颜色改为红色 |
| `b` | 将画笔颜色改为蓝色 |

### 编程要求

1. 每一个按键功能都应封装为函数；
2. 使用键盘监听功能；
3. 程序运行后，绘图窗口必须能够持续接收键盘输入。

### 可能使用

```python
screen.listen()
screen.onkey()
```

### 需要思考

为什么传给 `onkey()` 的通常是函数名，而不是函数调用结果？

例如：

```python
screen.onkey(move_forward, "Up")
```

而不是：

```python
screen.onkey(move_forward(), "Up")
```

# 文件命名规则

建议统一使用以下格式：

```text

```

例如：

```text

```

命名时注意：

1. 使用两位数字表示题号；
2. 使用小写英文字母；
3. 多个单词之间使用下划线；
4. 不使用空格；
5. 不使用中文；
6. 不要使用与 Python 标准库相同的名称。

```text

```

否则可能影响标准库模块的正常导入。

    main()
